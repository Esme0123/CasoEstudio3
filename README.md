# Proyecto: Clasificación Inteligente de Inventario mediante MobileNetV4

## Descripción
Este repositorio contiene el desarrollo del laboratorio de la Unidad 4 para del Grupo 5. El proyecto implementa un sistema de visión artificial aplicado al control de calidad y catalogación de productos en entornos industriales (Edge-Commerce).

## Objetivo
Comparar el rendimiento de una red convolucional (CNN) entrenada desde cero (Baseline) frente a un modelo de última generación (MobileNetV4, ECCV 2024) utilizando la técnica de Transfer Learning para superar las limitaciones de datos en entornos restringidos.

## Estructura del Repositorio
- `/data`: Dataset de entrenamiento y validación.
- `/notebooks`: Implementación paso a paso.
- `/src`: Scripts modulares (dataloader, modelos, Grad-CAM).
- `/models`: Pesos y arquitectura optimizada.

## Referencias Técnicas
1. **Survey:** Transfer Learning Applied to Computer Vision Problems (2024).
2. **Arquitectura:** MobileNetV4: Universal Models for the Mobile Ecosystem (ECCV 2024).
